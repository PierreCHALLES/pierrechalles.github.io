Projet RFTG - Partie Mobile

Description

Cette application mobile est conçue pour les clients du village vacances. Disponible sur Android et sur des bornes de location intégrées (tablettes), elle permet aux utilisateurs de parcourir le catalogue de DVD et de louer les films de leur choix. Les clients ont également la possibilité d’opter pour une livraison à domicile.

Fonctionnalités principales

  • Consultation du catalogue de DVD
  • Location de films via l'application mobile ou les bornes de location
  • Option de livraison à domicile
  • Interface simple et intuitive pour une expérience utilisateur optimale

Technologies utilisées

  • Java pour le développement Android
  • API RESTful pour la communication avec le serveur
  • Firebase pour l'authentification et la gestion des utilisateurs
