Projet RFTG - Partie Web

Description

Cette application est destinée à M. Emet Brown et à son équipe pour la gestion des activités liées aux DVD dans le village vacances. Accessible via un navigateur web, elle permet aux employés de gérer efficacement le stock de DVD. Les fonctionnalités incluent l’ajout, la suppression, la mise à jour et la recherche de DVD par titre, genre ou époque.

Fonctionnalités principales

  • Gestion des stocks de DVD (ajout, suppression, mise à jour)
  • Recherche de DVD par titre, genre, époque
  • Traçabilité des locations de DVD (historique des emprunts)
  • Interface d'administration sécurisée

Technologies utilisées

  • HTML, CSS, JavaScript pour le front-end
  • PHP, Java (Spring) pour le back-end
  • MySQL pour la gestion des bases de données
